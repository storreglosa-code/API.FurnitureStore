Preparacion para crear los ENDPOINTS de Registro y Login (VIDEO 33):
En el proyecto Shared creamos dos carpetas:
-AUTH 
    --> AuthResult.cs --> La vamos a usar en el controller de autenticación para devolver una respuesta. Objeto para transferir información entre capas.

 public class AuthResult
 {
     public string Token { get; set; }
     public bool Result { get; set; }
     public List<string> Errors { get; set; }
 }

-DTOs 
    --> UserRegistrationRequestDto --> Va a ser el request, o lo que vamos a pedir a quien quiera que consuma el endpoint de registro. 
    En el controlador vamos a validar si falta alguno de estos atributos --> devuelve BadRequest.

 public class UserRegistrationRequestDto
 {
     [Required]
     public string Name { get; set; }

     [Required]
     public string EmailAddress { get; set; }

     [Required]
     public string Password { get; set; }
 }
